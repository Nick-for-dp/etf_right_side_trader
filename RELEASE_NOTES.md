# 更新说明

## 版本主题

生产基线升级为 `baseline-v2.4`，并完成文档结构整理。

## 核心变更

1. 生产策略基线从 `v2.3 gate-lite + regime_group` 升级为 `baseline-v2.4`。
2. `baseline-v2.4` 固化以下口径：
   - 70/15/15 阶梯建仓加仓
   - 分市场/分波动止损 profile
   - 90 日最大持有
   - gate-lite + regime_group 门控
3. 生产风控支持：
   - `max_holding_days`
   - `profile_stop_loss`
   - A 股高波动 ETF 使用 ATR profile 止损
4. advisor 支持 70/15/15 加仓预算门控：
   - 第一次加仓要求浮盈且 score >= 60
   - 第二次加仓要求浮盈且 score >= 70
   - 不满足时降级为继续持有，来源为 `add_budget`
5. 新增因子消融和再入场实验框架，支持分市场/分波动报告。
6. 文档结构调整：
   - `README.md`：整理项目入口和 baseline-v2.4 摘要
   - 本地规划、实验笔记和架构归档保留在开发环境，不纳入远程仓库

## 关键实验结论

已采纳：

- 70/15/15 阶梯建仓加仓
- 分市场/分波动止损
- 90 日最大持有

不再推进：

- 止损后再入场硬过滤
- 复杂硬门控
- 舆情事件数据
- 行业资金流数据
- 基本面估值负向门控
- 分钟级或高频策略
- LLM 直接生成交易信号
